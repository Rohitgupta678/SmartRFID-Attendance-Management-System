The **Attendance Management System (RFID-Based)** is a Java Web Application designed to automate attendance marking using RFID technology. When students tap their RFID cards on the reader, their attendance is instantly recorded in the MySQL database. The system includes **Admin** and **Teacher** modules to manage students, track attendance, and generate detailed reports.
